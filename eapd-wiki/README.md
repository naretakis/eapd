## Clone the Wiki

```bash
$ git clone git://git@github.com:CMSgov/eAPD.wiki.git
```

## Run it locally

```bash
$ cd eAPD.wiki
$ brew install cmake
$ gem install gollum  # ruby is a requirement
$ gollum
$ open http://localhost:4567
```

[Gollum configuration](https://github.com/gollum/gollum#configuration)
